Angular Material Example

This Angular Material example is a single page Angular 6 application that is deployed on GitHub. The application was created using the Angular CLI and the Angular 6 Material code. Accessibility test results are included at the bottom of the page.

To improve accessibility, the following have been added to the application:

  • W3C WAI ARIA Roles
  • W3C WAI ARIA 14: Using aria-label to provide an invisible label where a visible label cannot be used
  • W3C WAI ARIA 21: Using aria-invalid to indicate an error field
  • Styles to improve color contrast.

Angular Material automatically includes the following on the form controls in this example:

  • mat-form-field uses the label tag to automatically create the aria-label for the input
  • mat-select automatically adds role="listbox" to the component and role="option"to the select options.
  • mat-radio-button automatically adds input type="radio" to the component.
  • mat-checkbox uses an internal input type="checkbox". This internal checkbox receives focus and is automatically labelled by the text content of the mat-checkbox element. In addition, aria-checked="false" is added and automatically changes to aria-checked="true" when the checkbox is checked.
  • tabindex="0"

Note: The Angular Material web site lists the built-in accessibility for each component.

Tested on:

  • Mac OS Sierra 10.13.4, VoiceOver, and the Chrome 66.0.3359.181 browser.
    • Results - VoiceOver announces:
      • titles, links, headings, aria-labels, and landmarks
      • lists and number of list items
      • dialog modal content and buttons
      • form controls, states (i.e. selected, checked, etc.) and required fields
      • form validation message on submit
  • Windows 7, JAWS 2018.1805.33 ILM, and the Chrome 67.0.3396.79 browser.
    • Results - JAWS announces:
      • titles, links, headings, aria-labels, and landmarks
      • lists and number of list items
      • dialog modal content and buttons
      • form controls, states (i.e. selected, checked, etc.) and required fields
      • form validation message on submit
      • Note: If a text input field has been visited and left empty, then revisted, Jaws announces: Last name, star edit required, invalid entry, type in text, and a red underline displays underneath the text input field.
